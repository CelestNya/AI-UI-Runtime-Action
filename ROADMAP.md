# Roadmap

## v0.1

Current public MVP:

- enable debug on the current page
- in-page overlay with `Position / Size / Describe`
- hover highlight and explicit selection
- move preview, resize preview, and manual request input
- structured intent output
- `Copy for AI` main path
- Chinese and English UI

## v0.2

Make targeting and playback more reliable:

- stronger hierarchy confidence scoring
- better parent-container promotion
- explicit group bounding box for multi-select
- undo / redo stack
- action history panel
- clearer prompt templates for common UI changes

## v0.3

Move from copied context to agent-ready integration:

- pluggable AI target abstraction
- send-to-Claude / Codex / Cursor adapters
- richer source-location hints from framework metadata
- intent batching and multi-step sessions

## v0.4

Connect runtime interaction to source-code change review:

- patch preview pipeline
- accept / reject code changes
- maintainable layout strategy hints
- patch traceability from runtime selection to source file

## v0.5+

Longer-term direction:

- layout-aware resize / spacing / alignment intents
- stronger support across React / Vue / generic DOM apps
- collaboration and shared review flows
- local agent bridge and tool-call orchestration
